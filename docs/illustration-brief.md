# Illustration brief — the Inner Circle character library

Generate these with ChatGPT (or any image model), then hand me the folder; I'll wire them in. Each file must be a **transparent-background PNG, 1024×1024**, named exactly `illo-<slot>.png`. Once added to the Xcode asset catalog they appear automatically — every slot already has a live placeholder in the app.

## Art direction (paste this into every generation)
> Flat vector character illustration in the style of Duolingo's mascot art: bold shapes, thick simple forms, no outlines or soft gradients, big expressive eyes, exaggerated poses, 2-3 tone shading max. Palette anchored on purple #7B45EB with accents of coral #FF7042, mint #30CC82, sky #1CB0F6, pink #FA6BB8, mango #FFB819. Friendly, slightly dramatic, a little unhinged — never corporate. Transparent background, centered subject, generous margins.

**The mascot**: design ONE recurring character first (suggestion: a round purple creature — a loop/ring motif nodding to "circle") and reuse it across all mascot-* slots so the app has a face. Generate the mascot reference sheet first, then feed it back for consistency.
